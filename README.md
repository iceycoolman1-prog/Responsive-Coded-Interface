# Project Title
The Character Tavern
## Project Description
What did you make?
I created a small website that would contain the story, image, and stats of a character created for Tabletop rpgs.
## Intended User or Player
Who is this interface designed for?
At the moment it would be designed for me and my friends, but I would hope to expand upon it and make it possible for other people to share their
characters.
## User / Player Goal
What should the user or player be able to do or understand?
The user should be able to understand that this is a themed website geared towards Dungeons and Dragons, and they should also understand that you can
select the character icons to showcase each character.
## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.
Baldur's Gate 3 - A good structure to showcase information
Team Fortress 2 - A nice way to organize the previews of characters
Divinity: Original Sin - Visual hierarchy and symmetry
## Design Choices
- Color: I wanted a darker background so the content could be held in beige boxes, emulating old paper
- Typography: I chose Blackadder ITC and Colonna MT for its balance of stylization and readability
- Visual hierarchy: I placed the items in a way that I felt would support the visual hierarchy. 
- Feedback states: I added a drop shadow to the character previews to signify that you can select them.
- Responsive design: I added navigational arrows for the character previews.
